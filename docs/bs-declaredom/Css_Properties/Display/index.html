Module Css_Properties.Display

Display

The display property is special: it overrides the default display style of an element. It's treated differently in order to keep things type safe.

Instead of providing a make function, this module provides a function for each display style that takes a ruleset and creates an overrided ruleset.

For example, block returns Css_Property.override_block Js.Dict.t. These types are used by Html_Nodes to provide override versions of each constructor.

none is the only function that's analogous to make, and creates a display: none property for general purpose use.

type +'a t = 'a Css_Property.t constraint 'a = [> Css_Property.display' ]
val none : unit -> [> Css_Property.display' ] t
